# 🧠 Natural Language Processing Coursework

This repository contains a collection of assignments completed as part of the **Natural Language Processing (NLP) course** in the B.Sc. Computer Science & Artificial Intelligence program at **Technische Hochschule Ingolstadt**.

The assignments explore practical NLP workflows including **text preprocessing, feature extraction, and machine learning-based text analysis** using Python and Jupyter Notebook.

# 🚀 Running the Project

### Clone the repository

```
git clone https://github.com/adilahmed08/Natural_Language_Processing_Coursework.git
cd Natural_Language_Processing_Coursework
```

### Install dependencies

```
pip install pandas numpy scikit-learn nltk jupyter
```

### Run notebooks

```
jupyter notebook
```

Open any assignment notebook to reproduce the results.
